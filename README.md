# Pi-ETNA

pi-etna is a pi extension for LLM-guided ETNA workload construction. Right now it's focused on Rust projects with plans to expand later.

Some existing workloads I've been working on:

- [im-rs](https://github.com/alpaylan/im-rs-etna)
- [petgraph](https://github.com/alpaylan/petgraph-etna)
- [arroy](https://github.com/alpaylan/arroy-etna)
- [roaring-rs](https://github.com/alpaylan/roaring-rs-etna)
